Este repositório reúne os principais projetos acadêmicos que desenvolvi durante a faculdade. Cada projeto foi criado com o objetivo de resolver problemas reais, aplicando conceitos aprendidos em sala de aula.

📁 Projetos

--- 📂 Gestor de Portfólio ---
> O gestor em questão tem a fucionalidade de listar livros com seu título, autor e um breve resumo.

Descrição: Sistema de gerenciamento de leitura pessoal via terminal, com persistência de dados em JSON.

Tecnologias: Python, JSON

Conceitos aplicados: CRUD completo, persistência de dados, histórico de alterações, tratamento de exceções, estruturação com funções, estatísticas de progresso
